# Consume SDK Typescript Project

## Create npm project
`npm init -y`

## Create typescript configuration file
`tsc --init`

## Create main app file `app.ts`

## Install or Link SDK dependency

- Install SDK
    `npm install sdk-tsc-basic`

- Link SDK
    `npm link ..\sdk-tsc-basic`


## Run 
```
tsc
node dist/app.js
```
